## Implementation Notes

1. **Reuse Canvas Component**: Same `<EditorCanvas>` component, just pass `mode="viewer"` prop
2. **Conditional Rendering**: Hide sidebars and tools when `mode="viewer"`
3. **Event Handlers**: Disable selection/editing in viewer mode, enable hover tooltips
4. **URL Structure**:
   - Editor: `/editor/:projectId`
   - Standalone Viewer: `/viewer/:mapId`
   - Embedded Viewer: `/embed/:mapId`
5. **Tab Switching**: In editor, clicking [Viewer] tab shows viewer preview without saving first (shows last saved state)
